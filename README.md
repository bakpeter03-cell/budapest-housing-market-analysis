# Budapest Housing Market Analysis

## Project Objective
This project analyzes the Budapest housing market at district level using housing listing data and official Hungarian housing statistics. The goal is to identify which districts are the most expensive, which are relatively more affordable, and which property characteristics are most strongly associated with price differences.

## Business Questions
- Which Budapest districts have the highest and lowest housing prices?
- Which districts have the highest and lowest price per square metre?
- How do housing prices differ across district groups?
- Which districts appear less affordable?
- Which property features are associated with higher asking prices?

## Project Structure
- `data/raw/` – original downloaded datasets
- `data/processed/` – cleaned datasets
- `notebooks/` – Jupyter notebooks for analysis
- `sql/` – SQL queries used for exploration
- `visuals/` – exported charts and figures
